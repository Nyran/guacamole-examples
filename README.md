Guacamole
=========

Installation
------------

You need [guacamole](https://github.com/vrsys/guacamole).

### Windows

We recommend building the examples with Visual Studio 2013. 
Configure with `cmake` and build the examples with Visual Studio.

### Linux

    ./scripts/make.sh
    cd examples/scenegraph
    ./scenegraph.out
